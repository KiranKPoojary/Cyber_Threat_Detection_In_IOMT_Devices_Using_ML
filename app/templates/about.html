About the IoMT Threat Detection System

Dataset Information

We used the CIC-IoMT2024 dataset, a comprehensive collection of real-world healthcare IoMT traffic. It contains benign and multiple categories of attacks, including:

  • DoS, DDoS, Mirai
  • Port scanning, Spoofing
  • Botnet, Reconnaissance

Features: Header-Length, IAT, Packet Sizes, Flags, and Statistical Features.

CNN Model

The Convolutional Neural Network model is trained on 2D feature representations extracted from network flows. It uses multiple convolution layers to detect patterns of malicious activity in traffic data.

  • Input: 2D matrix representing feature map
  • Layers: Conv2D, ReLU, Pooling, Dense
  • Output: Binary classification (0 = Benign, 1 = Attack)
