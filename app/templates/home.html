Project Overview

This project is focused on providing an intelligent security solution for Internet of Medical Things (IoMT) networks. It uses Deep Learning—specifically, a CNN (Convolutional Neural Network)—to detect whether incoming traffic is benign or malicious. Our platform ensures that real-time packet traffic from healthcare devices like ECGs, insulin pumps, and monitors is protected against cyber threats.

The system combines real-time packet capturing, feature engineering, and deep learning prediction into a single, lightweight application built with Flask and Socket.IO.

How It Works

  1. Capture network packets using Wireshark (Tshark)
  2. Extract features and preprocess data
  3. Convert traffic patterns into 2D format for CNN input
  4. CNN model classifies traffic as Benign or Attack
